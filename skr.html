SKR fees

Tips, protocol fee, treasury, and burn notes · Updated July 22, 2026

SKR is the Solana Mobile ecosystem token. Skroll is an independent app that uses SKR for tips and optional paid features. Skroll does not issue SKR and does not control Solana Mobile staking.

1. Tips

  • Users tip creators in SKR from a connected Seeker Wallet.
  • Default tip base and multipliers are set in the app config and can change over time.
  • Every tip is an on chain SPL transfer (plus a small SOL network fee paid by the sender).
  • Confirmed tips cannot be reversed by Skroll.

2. Protocol fee

Skroll takes a 10% protocol fee on each tip. The creator receives the remaining 90%.

  • Fee basis points: 1000 (10%).
  • Treasury wallet: qXhoL96gkzqe2KFMpVPseEWVzyzVUibw7YcXknUJQ85
  • Paid features such as retention extend or featured boost, when enabled, may also route a protocol share.

3. SKR burn

Skroll does not automatically burn tip fees today. The 10% protocol share is sent to the treasury wallet above for product operations, infrastructure, and future ecosystem programs.

If a burn path is enabled later (for example routing part of treasury inflows to a burn address), this page will be updated with the burn mint address, percentage, and effective date before it goes live.

4. No burner wallets

Skroll is built for Seeker Wallet / Seed Vault via Mobile Wallet Adapter. The app does not create disposable “burner” wallets, does not hold user keys, and cannot move funds without an explicit wallet signature.

5. Risks

  • SKR price can move quickly.
  • Wrong destination approvals are irreversible.
  • Network congestion can delay or fail transactions.
  • Always verify mint SKRbvo6Gf7GondiT3BbTfuRDPqLWei4j2Qy2NPGZhW3 before trading elsewhere.
